Кинематика

Прямолинейное движение:

1. s = v * t

2. V = S / t

3. x = x₀ + Sx = x₀ + Vt (Уравнение координат)

Прямолинейное равноускоренное движение:

1. a = (V - V₀) / t

2. V = V₀ + at

3. s = V₀t + (a * t²) / 2

4. s = (V² - V₀²) / 2a

5. x = x₀ + V₀t + (a * t²) / 2 (Уравнение координат)

Свободное падение тел:

1. a = g = 10 м/с²

2. v = gt

3. s = (g * t²) / 2 = h

4. y = h - (g * t²) / 2

Движение вверх:

1. vy = v₀ - gt

2. v₀ = gt - vmax

3. h = v₀t - (g * t²) / 2

4. y = y₀ + v₀t - (g * t²) / 2

Движение по окружности:

1. a = v² / r

2. w = Δφ / T

3. T = [e]

4. T = 2Пr / v

5. D = 1 / T = V / 2Пr
