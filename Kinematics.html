Кинематика

Прямолинейное движение:

1. s = v * t

2. V = S / t

3. x = xo + Sx = xo + Vt (Уравнение координат)

Прямолинейное равноускоренное движение:

1. a = (V - Vo) / t

2. V = Vo + at

3. s = Vot + (a * t²) / 2

4. s = (V² - Vo²) / 2a

5. x = xo + Vot + (a * t²) / 2 (Уравнение координат)

Свободное падение тел:

1. a = g = 1o м/с²

2. v = gt

3. s = (g * t²) / 2 = h

4. y = h - (g * t²) / 2

Движение вверх:

1. vy = vo - gt

2. vo = gt - vmax

3. h = vot - (g * t²) / 2

4. y = yo + vot - (g * t²) / 2

Движение по окружности:

1. a = v² / r

2. w = Δφ / T

3. T = [e]

4. T = 2Пr / v

5. D = 1 / T = V / 2Пr

Движение тел под углом и горизонту:

1. vox = vocos(α)

2. L = vxt = + vocos(α)t

3. x = xo+ vocos(α)t

4. voy = vosin(α)

5. vy = vosin(α) - gt

7. s = vosin(α)t - (g * t²) / 2

7. t = 2vosin(α) / g

8. V = √(vx² + voy²)

9. xmax = (vo²sin(2α)) / g

Движение тел брошенного горизонтально:

1. vx = vo

2. l = voxt = vot

3. vy = gt

4. h = (g * t²) / 2

5. y = yo - (g * t²) / 2
